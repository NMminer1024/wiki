# LED 指示灯

带板载 RGB / 状态 LED 的板子会用它做实时心跳指示。

| 状态                  | LED 表现                                |
| --------------------- | --------------------------------------- |
| 正在连接 WiFi         | 呼吸灯                                   |
| 挖矿中                | 慢速脉动，同步 share 提交                |
| 接受 share            | 短促亮闪                                 |
| 拒绝 share            | 短促暗闪                                 |
| 错误                  | 持续单色（与板型相关）                   |

如果嫌 LED 晃眼，可在配网门户里 **禁用**。
