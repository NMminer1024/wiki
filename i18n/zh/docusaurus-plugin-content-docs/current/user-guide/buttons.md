# 按键与触摸

不同的板子有的只有一个按键（`BOOT`），有的两个（`BOOT` + `USER`），有的是触摸屏。NMMiner 在三种布局下都给出一致的手势。

## 单按键板

| 手势       | 动作                                             |
| ---------- | ------------------------------------------------ |
| 单击       | 唤醒屏幕（带屏板）。                             |
| 双击       | 切到下一屏。                                     |
| 长按       | 进入矿机配置（强制 AP 模式）。                   |

## 双按键板

| 按键    | 手势       | 动作                                              |
| ------- | ---------- | ------------------------------------------------- |
| `BOOT`  | 单击       | 唤醒屏幕。                                        |
| `BOOT`  | 双击       | 唤醒屏幕。                                        |
| `BOOT`  | 长按       | 清空所有保存的状态（如果固件构建里开启此特性）。  |
| `USER`  | 单击       | 唤醒屏幕。                                        |
| `USER`  | 双击       | 切到下一屏。                                      |
| `USER`  | 长按       | 进入矿机配置（强制 AP 模式）。                    |

## 触摸屏板

自 v2.0.01 起，触摸屏板（NM-TV-154、CYD 2.4 / 2.8 / 3.5、Waveshare 3.5 触摸等）支持 **直接在屏幕上** 操作：

| 手势                       | 动作                                              |
| -------------------------- | ------------------------------------------------- |
| 轻点                       | 切到下一屏（屏幕睡眠时同时唤醒）。               |
| **长按 10 秒**             | 进入矿机配置（强制 AP 模式）。                    |

屏幕页顺序见 [屏幕页面](./screens.md)，长按后的行为见 [配网门户](./configuration-portal.md)。
