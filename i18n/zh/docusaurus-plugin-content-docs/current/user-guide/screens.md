# 屏幕页面

带屏板会在一组固定页面之间循环切换。切换手势见 [按键与触摸](./buttons.md) — 视硬件可以是 tap、双击或滑动。

NMMiner 内置以下原生分辨率：

```
128x64    128x128    160x80     240x135
240x240   320x170    320x172    320x240
480x320   536x240
```

## 页面

| 顺序 | 页面       | 用途                                                                       |
| ---- | ---------- | -------------------------------------------------------------------------- |
| 1    | **Loading**| 开机界面 + 启动状态、固件版本、网络握手。                                   |
| 2    | **Config** | AP 模式时显示 — 含 QR / IP / SSID 指引你进入 captive portal。              |
| 3    | **Miner**  | 实时 hashrate、**session 最佳** + **历史最佳** 难度、accepted / rejected、矿池 URL、运行时长。**主工作页**。 |
| 4    | **Clock**  | 大时钟 + 日期，BTC 价格跑马灯（涨绿跌红）。可选 12 / 24 小时制 + 可配置日期格式。 |
| 5    | **Price**  | 两种模式：<br/>• **K 线** — 主币烛台图，间隔可配（`15m` / `1h` / `4h` / `1d` …）。<br/>• **Wall** — 多币种价格墙（最多 20 个币对）。 |
| 6    | **Weather**| 当前天气 + AQI + 3 天预报 + 日出日落。可选温度单位（°C / °F）、风速单位（km/h / mph / m/s）、高度计模式（气压 / 湿度）。 |
| 7    | **Ladder** | **可选** 的全球 top-10 排行榜，按历史最佳 share 难度排序。钱包按 `first4…last4` 显示保护隐私。默认关闭。同一份数据也发布在 [ladder.nmminer.com](https://ladder.nmminer.com/)，详见 [天梯榜](./ladder.md)。 |

:::info
所有板子页面顺序一致；小屏 / OLED 跳过装不下的页面。**断电重开后会自动回到上次停留的页面**（v1.8.20 起）。
:::

## 屏保

矿机空闲足够久会进入屏保：

- **图片模式** — 轮播通过 NM Monitor 上传的自定义图片。
- **黑屏模式** — 完全关掉背光（或 OLED 调到最低对比度），最省电、**hashrate 最高**。

唤醒手势：单击 `BOOT`（或 `USER`），或触摸屏轻点。详见 [屏保](./screen-saver.md)。
