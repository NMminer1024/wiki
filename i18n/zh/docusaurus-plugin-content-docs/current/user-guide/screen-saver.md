# 屏保

空闲一段时间后屏幕进入屏保。

## 模式

| 模式    | 效果                                                                                      |
| ------- | ----------------------------------------------------------------------------------------- |
| `IMAGE` | 轮播通过 NM Monitor 上传的自定义图片，带淡入淡出过渡。                                  |
| `BLACK` | 关掉背光（OLED 调到最低对比度），最省电，**hashrate 最高**。                            |

模式可以在 [NM Monitor](./nm-monitor.md) 的 **Preferences** 板块**实时切换** — 不需要重启。

## 自定义图片

NM Monitor 的 **Screensaver** 页面让你直接从浏览器上传 PNG / JPG。浏览器端先把图转成 NMMiner 私有的 `.ss` 格式（RGB565 + RLE），然后推送到设备 `/ss/` 目录。

- 图片**自动匹配当前屏幕分辨率**。
- 每个 `.ss` 文件最大 **200 KB**。
- 设备自管文件系统 — 满了会自动覆盖最旧的屏保。

想脚本化上传？见 [API › 屏保上传](../api/screensaver-upload.md)。

:::tip
追求最高 hashrate 选 **Black** 模式 — 屏幕刷新会吃掉一部分本可以用来计算 hash 的 CPU 时间。
:::
