# ESP32-C5 开发板

ESP32-C5 是较新的双频 RISC-V 芯片。NMMiner 支持包括定制的 NM-CYD-C5 开发板。

| 开发板                  | Hashrate         | 芯片      |
| ----------------------- | ---------------- | --------- |
| ESP32-C5 DevKitC-1      | **150.2 KH/s**   | ESP32-C5  |
| NM-CYD-C5               | **150.2 KH/s**   | ESP32-C5  |

:::info
C5 支持仍在持续优化。最新数字关注 [GitHub release log](https://github.com/NMminer1024/NMMiner#release-log)。
:::
