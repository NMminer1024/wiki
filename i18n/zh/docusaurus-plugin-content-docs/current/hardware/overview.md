# 硬件总览

NMMiner 覆盖四个芯片家族，每族都有各自的峰值算力和支持板列表。

| 芯片家族    | 典型峰值算力              | 说明 |
| ----------- | ------------------------- | ---- |
| **ESP32-D0**| 最高 **1,035 KH/s**       | 原版 ESP32，性能最强。 |
| **ESP32-S3**| 最高 **400 KH/s**         | 向量 ISA，多搭 TFT / AMOLED 屏。 |
| **ESP32-C3**| 最高 **402 KH/s**         | 单核 RISC-V，最便宜。  |
| **ESP32-C5**| 最高 **150 KH/s**         | 新的双频 RISC-V 芯片。 |

在子页面里找你那块板子：

- [ESP32-D0 开发板](./esp32-d0-boards.md)
- [ESP32-S3 开发板](./esp32-s3-boards.md)
- [ESP32-C3 开发板](./esp32-c3-boards.md)
- [ESP32-C5 开发板](./esp32-c5-boards.md)

:::tip
表格里的 hashrate 是在 **真机** 上、**最高稳定的 UI 刷新档** 下、**Black 屏保模式** 测出来的。降低屏幕刷新还能在每块板上再多榨出几 KH/s。
:::
