# NMMiner Wiki

欢迎来到 **NMMiner** 的用户 & 集成开发者文档中心。

**NMMiner** 是一款高性能固件，可以把常见的 ESP32 开发板变成一台比特币（以及 DigiByte）Stratum 矿工。支持 ESP32-D0、ESP32-S3、ESP32-C3、ESP32-C5 四个芯片家族，单机峰值算力：

- **ESP32-D0** — 最高 **1,035 KH/s**
- **ESP32-S3** — 最高 **400 KH/s**
- **ESP32-C3** — 最高 **402 KH/s**
- **ESP32-C5** — 最高 **150 KH/s**

## 你可以用 NMMiner 做什么

- 把一块便宜的 ESP32 板烧上固件，几分钟内就能在 **NMMiner 自家的 SoloBTC 矿池** 上挖矿 — 固件出厂已配置好。
- 用手机或电脑通过 captive portal 修改矿池、WiFi、钱包、屏幕等参数。
- 在板载屏幕上实时看到 hashrate、矿池地址、份额数、BTC / 多币种价格图表、天气等。
- 直接在浏览器打开 **NM Monitor** 来管理设置、上传自定义屏保，并通过 **Swarm 菜单** 一眼看到整张局域网内所有矿工的总算力。
- 基于设备公开的 HTTP API 自己写 dashboard 或自动化脚本。
