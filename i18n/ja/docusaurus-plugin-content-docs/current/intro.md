# NMMiner Wiki

**NMMiner** のユーザー＆インテグレーター向けドキュメントハブへようこそ。

**NMMiner** は、一般的な ESP32 開発ボードを Bitcoin（および DigiByte）Stratum マイナーに変える高性能ファームウェアです。ESP32-D0、ESP32-S3、ESP32-C3、ESP32-C5 チップファミリーに対応しており、デバイス上でのピークハッシュレートは以下の通りです：

- **ESP32-D0** — 最大 **1,035 KH/s**
- **ESP32-S3** — 最大 **400 KH/s**
- **ESP32-C3** — 最大 **402 KH/s**
- **ESP32-C5** — 最大 **150 KH/s**

## NMMiner でできること

- 低コストの ESP32 ボードにフラッシュし、数分で **NMMiner 独自の SoloBTC プール** でハッシュ計算を開始できます — ファームウェアは初期設定済みで出荷されます。
- スマートフォンや PC のキャプティブポータルから、プールの変更、WiFi の切り替え、ウォレットや画面オプションの設定が可能です。
- デバイス画面上で、ライブハッシュレート、プール URL、シェア統計、BTC / マルチコイン価格チャート、天気を表示できます。
- 任意のブラウザで **NM Monitor**（マイナー自身が配信）を開き、設定の管理、カスタムスクリーンセーバーのアップロード、**Swarm** メニューによるローカルフリート全体の監視が行えます。
- ドキュメント化された HTTP API を使って、独自のダッシュボードや自動化を構築できます。
