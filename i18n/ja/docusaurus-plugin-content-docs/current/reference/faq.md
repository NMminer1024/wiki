# よくある質問

## NMMiner が単独で Bitcoin ブロックを見つけることはありますか？

数学的には、はい — 宝くじ券と同じ確率で。現実的には、いいえ。ポイントは **楽しさ**（とネットワークのサポート）であり、報酬ではありません。

## 消費電力はどのくらいですか？

ほとんどのサポートボードは 1 W 未満です。$0.30 /kWh でも、月額 30 セントをはるかに下回ります。

## NMMiner がデフォルトで使用するプールはどれですか？

NMMiner は独自の SoloBTC プールに事前設定された状態で出荷されます：

- プライマリ: `stratum+tcp://solobtc.nmminer.com:3333`
- フォールバック: `stratum+tcp://au.solobtc.nmminer.com:3333`

**ウォレットアドレスのみ** を自分の BTC アドレスに変更する必要があります。プール URL は通常そのままで問題ありません。

## 別のプールを使用できますか？

はい — Stratum v1 を話すものなら何でも動作します。コミュニティの SoloBTC および DGB オプションについては [プールリスト](./pool-list.md) を参照してください。高い最小シェア難易度を義務付ける大手商用プールはすべてのシェアを拒否します — NMMiner の公式 [SoloBTC](https://solobtc.nmminer.com) プールまたはソロスタイルのエンドポイントを使用してください。

## NMMiner は Bitcoin 以外をマイニングできますか？

はい — **DigiByte (DGB)** もサポートされています。DGB は同じ SHA-256d アルゴリズムを使用するため、同じファームウェアで動作します。プール URL を DGB プールに向け、DGB ウォレットアドレスを使用するだけです。

## デバイス上にウォレットは必要ですか？

いいえ。必要なのは **支払い先アドレス** だけです。プールは設定された BTC / DGB アドレスに直接報酬を送金します。

## 画面が数秒ごとに自動的に起動します。

それは無害なバックグラウンドリフレッシュです。ディスプレイのアクティビティをゼロにしたい（かつ最大ハッシュレートを得たい）場合は、**Black** スクリーンセーバーモードを使用してください。

## 同じ WiFi で 2 台のマイナーを実行できますか？

はい。NM Monitor の **Swarm** メニューは、同じ LAN 上のすべての NMMiner を自動検出します — /24 サブネットあたり最大 255 デバイス。[Swarm](../user-guide/swarm.md) を参照。

## マイナーが Loading 画面で止まっている / プールに接続できない

このチェックリストを実行してください：

1. **WiFi は 2.4 GHz である必要があります。** ほとんどの ESP32 チップは 5 GHz をサポートしていません。家庭の WiFi が同じ SSID で両方のバンドを持っている場合は、一時的に分離するか、明示的に 2.4 GHz の方に接続してください。
2. **ウォレットアドレスを確認してください。** ここにタイプミスがあると、プールがワーカーを拒否します。有効な Bitcoin アドレスは以下で始まります：
   - `bc1q...` — Native SegWit（現在最も一般的）
   - `bc1p...` — Taproot
   - `1...`    — レガシー P2PKH
   - `3...`    — P2SH（マルチシグ / ラップド SegWit）

   これ以外は有効な BTC アドレスでは **ありません** — 完全な文字列をコピーしたか再確認してください。
3. **プール URL を確認してください。** 箱出し状態では NMMiner は `stratum+tcp://solobtc.nmminer.com:3333` で事前設定されており、変更なしで動作します。プールを切り替えた場合、URL は `stratum+tcp://host:port`（または TLS の場合は `stratum+ssl://host:port`）の形式である必要があります。既知の正常なエンドポイントについては [プールリスト](./pool-list.md) を参照してください。
4. **プール難易度。** 高い最小シェア難易度を義務付ける大手商用プールは、すべての NMMiner シェアを拒否します。NMMiner の公式 [SoloBTC](https://solobtc.nmminer.com) プールを使用してください。
5. **電源。** 不安定な USB ポートはハングアップのように見えるブラウンアウトを引き起こす可能性があります。別のケーブル + 5 V/1 A 電源を試してください。

上記のいずれでも解決しない場合、マイナーは約 15 秒後に AP モードにフォールバックするはずです — 再度 `nmap-2.4g` に接続して値を再確認してください。

## 公開 API はありますか？

はい — すべてのデバイスはポート 80 で文書化された HTTP API を公開しています。[API リファレンス](../api/overview.md) を参照。

## ソースコードは利用可能ですか？

NMMiner ファームウェアはクローズドソースです。各デバイスの HTTP API がサポート対象の統合面です。付属の CLI フラッシュツールと SoloBTC プールのウェブフロントエンドは公開されています。

## Ladder ページは何をするのですか？

これは、参加マイナーが生成した過去最高のシェア難易度でランク付けされた **オプトイン** のグローバルトップ 10 リーダーボードです。ウォレットはプライバシーのため `first4…last4` 形式で表示されます。デフォルトでは無効です — NM Monitor の **Preferences** セクションから有効にしてください。公開ウェブビューは [ladder.nmminer.com](https://ladder.nmminer.com/) にあります — 詳細は [ラダー](../user-guide/ladder.md) を参照。
