# トラブルシューティング

## フラッシュツールにボードが表示されない

- **USB ケーブルを数回抜き差ししてください** — WebSerial ピッカーは、特にドライバインストール直後の最初のホットプラグでデバイスを見逃すことがあります。
- **データ** USB ケーブルを使用してください（給電専用ではないもの）。
- 別の USB ポートを試してください。一部のハブは CDC を通過させません。
- Windows では、ボードのブリッジチップに応じて **CP210x** または **CH340** ドライバをインストールしてください。
- `RESET` を押しながら `BOOT` を押し続けて、ダウンロードモードを強制してください。

## Loading で止まっている

- WiFi が **2.4 GHz** であることを確認してください（ほとんどの ESP32 チップは 5 GHz に対応していません）。
- 15 秒待ってください。接続できない場合、自動的に AP モードにフォールバックするはずです。

## AP `nmap-2.4g` が表示されない

- ボードの電源を入れ直してください。
- 設定ボタンを 5 秒間押し続けてください（[ボタン](../user-guide/buttons.md) を参照）。

## キャプティブポータルは表示されるが Save が機能しない

- 15 秒のフルウィンドウを待ってください — ストレージのフラッシュは意図的に遅くなっています。
- ページが戻らない場合は、電源を入れ直して再試行してください。

## プールが「Connecting」で止まっている

- プール URL とポートを確認してください。
- `stratum+ssl://` の代わりに `stratum+tcp://` を試してください（またはその逆）。
- [フォールバックプール](../user-guide/pools.md) を試してください。

## ハッシュレートがスペックシートより大幅に低い

- **UI リフレッシュ** 設定は小さな CPU では重いです。下げてみてください（フォームでは高い値）。
- **Black** スクリーンセーバーを使用して、ディスプレイの再描画コストを完全に除去してください。
- 電源が 500 mA を持続できることを確認してください — 不安定な USB ポートはブラウンアウトを引き起こします。

## シェアがすべて拒否される

- プール難易度が高すぎます。NMMiner の公式 [SoloBTC](https://solobtc.nmminer.com) プール（または公式フォールバックプール）に切り替えてください。
- ウォレットフィールドを確認してください — 多くのプールは不正な形式のウォレットアドレスを拒否します。有効な Bitcoin アドレス形式については [FAQ](./faq) を参照。

## NM Monitor に到達できない

- あなたとマイナーが **同じ WiFi** にいることを確認してください。
- マイナーの Miner ページで IP を確認し、それを直接試してください。
- スマートフォン / PC の **VPN** を無効にしてください。一部の VPN は LAN トラフィックをブロックします。
- ルーターが **AP アイソレーション** を使用している場合、デバイスは互いに到達できません — そのモードを無効にしてください。

## Swarm メニューに実際より少ないマイナーしか表示されない

- **Re-scan** をクリックしてください（またはページをリロード）。
- LAN スキャンは 1 つの /24 サブネットのみを対象とします。異なるサブネットのマイナーは表示されません。
- 表示されないマイナーが画面の **Miner ページ** にいることを確認してください — 検出スレッドはそのページでのみ実行されます。
