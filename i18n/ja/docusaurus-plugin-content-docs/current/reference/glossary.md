# 用語集

| 用語                    | 定義                                                                       |
| ----------------------- | -------------------------------------------------------------------------------- |
| **AP モード**             | NMMiner が WiFi に到達できない場合にフォールバックする Soft-AP キャプティブポータル。 |
| **CYD**                 | 「Cheap Yellow Display」 — 低コストの ESP32 + TFT ボードのファミリー。                |
| **DGB / DigiByte**      | SHA-256d アルトコイン。NMMiner は箱出しで DGB をサポート — 同じファームウェア、異なるプール。 |
| **ハッシュレート**            | 1 秒あたりのハッシュ数。NMMiner では KH/s で報告。                                  |
| **ジョブ**                 | プールから配信される作業単位（`mining.notify`）。                          |
| **ラダー**              | [ladder.nmminer.com](https://ladder.nmminer.com/) で公開されデバイス画面にもミラーされる、オプトインのグローバルトップ 10 難易度リーダーボード。[ラダー](../user-guide/ladder.md) を参照。 |
| **ライセンス**             | ボードをファームウェアイメージに紐付けるデバイスごとのアクティベーション文字列。                  |
| **NM Monitor**          | すべてのマイナーからポート 80 で配信されるブラウザベースの UI。                          |
| **ノンス**               | マイナーがシェアを探索する際に反復する 32 ビットフィールド。              |
| **プール難易度**           | プールが受け入れる最小シェア難易度。                                       |
| **プローブ**               | LAN 識別に使用される軽量な `/probe` エンドポイント。                   |
| **シェア**               | プール難易度を満たすがネットワーク難易度は満たさないハッシュ。                        |
| **SoloBTC**             | NMMiner の公式マイニングプール: [solobtc.nmminer.com](https://solobtc.nmminer.com)。出荷されるすべてのデバイスのデフォルトプール。 |
| **Stratum**             | NMMiner が話すプールプロトコル。v1 のみ。                                       |
| **Swarm**               | NM Monitor 内の Swarm メニュー — LAN 全体のマイナー集約、インストール不要。/24 サブネットあたり最大 255 デバイス。 |
