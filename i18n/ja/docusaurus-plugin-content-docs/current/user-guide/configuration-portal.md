# 設定ポータル

設定ポータルは、AP モード中にマイナー自身から配信される小さなキャプティブポータル形式のウェブページです。

## ポータルが表示されるタイミング

- フラッシュ後の **初回起動**。
- **再接続失敗** — 保存された WiFi に約 15 秒以内に到達できない場合、自動的に AP モードにフォールバックします。
- **手動強制** — 設定ボタンを長押し（[ボタン](./buttons.md) を参照）。

## アクセス方法

1. スマートフォンまたはラップトップの WiFi 設定を開きます。
2. オープンホットスポット **`nmap-2.4g`** に接続します。
3. スマートフォンではキャプティブポータルブラウザが自動的にポップアップします。PC では `http://192.168.4.1` を開きます。

[初回設定](../getting-started/first-configuration.md) で説明されている設定フォームが表示されます。

## Save の動作

**Save** を押すと、新しい設定を書き込み、デバイス内ストレージのフラッシュを待ち（10〜15 秒）、その後再起動します。**このウィンドウ中に電源を切らないでください** — 自動的に再起動するのを待ってください。
