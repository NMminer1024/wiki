# LED インジケーター

ボード上の RGB またはステータス LED を搭載したボードは、それをリアルタイムのマイナーハートビートとして使用します。

| 状態           | LED パターン                          |
| --------------- | ------------------------------------ |
| WiFi 接続中 | 呼吸するような点滅                            |
| マイニング中          | シェア送信に同期したゆっくりとしたパルス |
| シェア承認  | 素早い明るいフラッシュ                   |
| シェア拒否  | 素早い減光 / 消灯フラッシュ                |
| エラー           | 単色点灯（モデル依存）       |

LED は、光が気になる場合に設定ポータルで **無効** にできます。
