# ファームウェアアップデート

アップグレードは最初のフラッシュと同じです：ブラウザフラッシュツールを再度開き、ボードを選び、インストールします。

## 手順

1. Chrome または Edge で [flash.nmminer.com](https://flash.nmminer.com) を開きます。
2. ボードプロファイルを選択します。
3. **Install** をクリックし、シリアルポートを選び、完了するまで待ちます。

アップデート後、**ライセンスを再有効化する必要はありません**。設定（SSID / プール / ウォレット）はアップグレード後も保持されます。

## ファームウェアバージョンのリモート追跡

現在のファームウェアバージョンは、すべてのマイナーから [`GET /api/system/info`](../api/system.md) を通じて報告されます。これを [Swarm](./swarm.md) 検出と組み合わせると、LAN 上のどのマイナーがまだアップデートが必要か一目で確認できます。

## バージョン間の変更点

完全な変更ログについては [GitHub リリースログ](https://github.com/NMminer1024/NMMiner#release-log) を参照してください。メジャーバージョンの引き上げ（例：**v1.x → v2.0**）では、NM Monitor と Swarm に破壊的な改善が導入されました — 古いリリースは新しい Swarm プロトコルと相互運用できないため、フリート全体を同じマイナーシリーズに保つことが最善です。
