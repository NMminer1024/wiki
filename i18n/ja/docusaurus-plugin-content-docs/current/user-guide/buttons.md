# ボタンとタッチ

ボードによって、ボタンが 1 つ（`BOOT`）、2 つ（`BOOT` + `USER`）、またはタッチスクリーンがあります。NMMiner は 3 つすべてに一貫したジェスチャーを提供します。

## 1 ボタンボード

| ジェスチャー       | アクション                                            |
| ------------- | ------------------------------------------------- |
| シングルクリック  | 画面を起動（ディスプレイ付きボード）。       |
| ダブルクリック  | 次の画面ページに切り替え。                   |
| 長押し    | マイナー設定に入る（AP モードを強制）。        |

## 2 ボタンボード

| ボタン   | ジェスチャー       | アクション                                                  |
| -------- | ------------- | ------------------------------------------------------- |
| `BOOT`   | シングルクリック  | 画面を起動。                                        |
| `BOOT`   | ダブルクリック  | 画面を起動。                                        |
| `BOOT`   | 長押し    | NVS からすべての保存状態をクリア（ビルドで有効な場合）。  |
| `USER`   | シングルクリック  | 画面を起動。                                        |
| `USER`   | ダブルクリック  | 次の画面ページに切り替え。                         |
| `USER`   | 長押し    | マイナー設定に入る（AP モードを強制）。              |

## タッチスクリーンボード

v2.0.01 以降、タッチボード（NM-TV-154、CYD 2.4 / 2.8 / 3.5、Waveshare 3.5 タッチなど）は **画面上で直接** ジェスチャーをサポートします：

| ジェスチャー                       | アクション                                            |
| ----------------------------- | ------------------------------------------------- |
| タップ                           | 次の画面ページに切り替え（スリープ中は画面の起動も行います）。 |
| **10 秒間の長押し**     | マイナー設定に入る（AP モードを強制）。        |

ページの順序については [画面](./screens.md) を、長押し後の動作については [設定ポータル](./configuration-portal.md) を参照してください。
