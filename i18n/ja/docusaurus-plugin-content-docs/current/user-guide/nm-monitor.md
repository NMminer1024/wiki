# NM Monitor

**NM Monitor** は、**すべての NMMiner デバイス内部** に存在するウェブアプリです。スマートフォンや PC にインストールするものは何もありません — 任意のブラウザでマイナーの IP を開くだけです。すべての監視とフリート管理は NM Monitor とその [Swarm](./swarm.md) メニューを通じて行います。

## NM Monitor を開く

1. スマートフォンまたは PC をマイナーと **同じ WiFi** に接続します。
2. マイナーの IP を見つけます — 最も簡単な方法は、デバイス画面の **Miner ページ** を確認することです（IP はステータスバーに表示されます）。または、ルーターの DHCP テーブルで調べます。
3. **Chrome、Edge、Safari、Firefox** で `http://<miner-ip>/` を開きます。覚えるべきポートはありません — ポート 80 です。
4. ホームページが自動的に読み込まれます。

:::tip
`http://<miner-hostname>/`（例：`http://nm-cyd-c5-ab12cd/`）をブックマークしてください — ほとんどの家庭用ルーターでは、ホスト名でもマイナーに到達できます。
:::

## NM Monitor でできること

| セクション              | 目的                                                          |
| -------------------- | ---------------------------------------------------------------- |
| **Dashboard**        | ライブハッシュレート、承認 / 拒否シェア、プール難易度、稼働時間、RSSI、空きヒープ、温度（サポート対象の場合）。 |
| **Swarm**            | LAN 全体の集約：同じ WiFi 上のすべての NMMiner を表示、ハッシュレート合計、特定のマイナーを点滅させる Ping。[Swarm](./swarm.md) を参照。 |
| **Network**          | ホスト名、WiFi SSID、WiFi パスワードの変更。                       |
| **Mining**           | プライマリ / フォールバックプール URL、ウォレットアドレス、プールパスワード。      |
| **Time**             | タイムゾーン、時刻形式（12 / 24 h）、日付形式。                 |
| **Preferences**      | 画面の明るさ、回転、LED 有効、スクリーンセーバータイムアウト、スクリーンセーバーモード（image / black）、[ラダー](./ladder.md) オプトイン。 |
| **Market**           | Price ページに表示するメインコインとウォッチリストの選択。   |
| **Weather**          | Weather ページで使用する都市 / 緯度 / 経度と単位の設定。     |
| **Screensaver**      | ブラウザ内で変換されマイナーにプッシュされるカスタム PNG のアップロード。 |
| **System**           | ファームウェアバージョン、ボード、空きヒープ、**Restart** ボタン。          |
| **API Doc**          | ライブ HTTP API リファレンス（`/api-doc` でも公開）。          |

## LAN 上のどこからでも読み取り可能

NM Monitor はすべての API エンドポイントで `Access-Control-Allow-Origin: *` を設定します。つまり、任意のオリジンから配信されたブラウザページがマイナーのステータスを読み取ることができます — 独自のダッシュボード構築に便利です。エンドポイントの完全なセットは [API リファレンス](../api/overview.md) に文書化されています。
