# ESP32-C5 ボード

ESP32-C5 は新しいデュアルバンド RISC-V チップです。NMMiner は専用の NM-CYD-C5 開発ボードを含むサポートを提供します。

| ボード                  | ハッシュレート       | チップ      |
| ---------------------- | -------------- | --------- |
| ESP32-C5 DevKitC-1     | **150.2 KH/s** | ESP32-C5  |
| NM-CYD-C5              | **150.2 KH/s** | ESP32-C5  |

:::info
C5 サポートは引き続きハッシュレートの改善が行われています。最新の数値については [GitHub リリースログ](https://github.com/NMminer1024/NMMiner#release-log) を参照してください。
:::
