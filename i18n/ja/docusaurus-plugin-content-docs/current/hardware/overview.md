# ハードウェア概要

NMMiner は 4 つのチップファミリーを対象としています。各ファミリーには独自のピークハッシュレートと、厳選されたサポートボード一覧があります。

| チップファミリー | 典型的なピークハッシュレート | 備考 |
| ----------- | --------------------- | ----- |
| **ESP32-D0**| 最大 **1,035 KH/s**  | 最高の生性能。オリジナル ESP32 ファミリー。 |
| **ESP32-S3**| 最大 **400 KH/s**    | ベクトル ISA、デュアルコア、通常 TFT / AMOLED ディスプレイと組み合わされます。 |
| **ESP32-C3**| 最大 **402 KH/s**    | シングル RISC-V コア、最も安価なティア。 |
| **ESP32-C5**| 最大 **150 KH/s**    | 新しいデュアルバンド RISC-V パーツ。 |

サブページでお手持ちの正確なボードを見つけてください：

- [ESP32-D0 ボード](./esp32-d0-boards.md)
- [ESP32-S3 ボード](./esp32-s3-boards.md)
- [ESP32-C3 ボード](./esp32-c3-boards.md)
- [ESP32-C5 ボード](./esp32-c5-boards.md)

:::tip
ハッシュレート列は、スクリーンセーバーを **Black** モードにし、最も高い安定した UI リフレッシュ設定で**実際のハードウェア上で測定**されたものです。画面リフレッシュ間隔を下げると、すべてのボードで数 KH/s の追加が得られます。
:::
