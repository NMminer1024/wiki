# ソロマイニング vs プールマイニング — どちらがあなたに適していますか？

マイニングに不慣れな方なら、**ソロマイニング** と **プールマイニング** という 2 つの言葉を耳にしたことがあるでしょう。これらはブロック報酬を得るための 2 つの異なる戦略であり、適切な方を選ぶことが非常に重要です。平易な言葉で分解してみましょう。

---

## 宝くじのアナロジー

Bitcoin マイニングを、約 10 分ごとに当選者が出る巨大な世界規模の宝くじだと想像してください。

- **賞金**: 3.125 BTC（現在のブロック報酬）に取引手数料を加えたもの。
- **チケット**: ネットワーク難易度を満たす有効なハッシュ。
- **オッズ**: あなたのハッシュレートをネットワーク全体のハッシュレートで割った値に比例。

このイメージを念頭に、2 つの戦略がどう異なるかを見ていきます。

---

## ソロマイニング — 一人で宝くじを買う

単独でマイニングします。マイナーは完全な Bitcoin ノード（または報酬を分割しない特別なソロマイニングプール）と直接通信します。独立してブロックを探索します。

あなたのマイナーが有効なブロックを見つけた場合、**報酬の 100% を受け取ります**。共有もプール手数料もありません。

**素晴らしく聞こえますよね？しかし落とし穴があります：**

> 典型的な NMMiner のハッシュレート（約数百 KH/s）では、Bitcoin ネットワーク上でブロックを見つけるまでの平均時間は **数世紀** 単位です。あなたは本質的に、一生のうちに当たらないかもしれない宝くじを引いているのです。

### メリット

- ヒットすればブロック報酬（3.125+ BTC）の **全額** を保持できます。
- 完全な独立性 — 信頼すべきプール運営者なし。
- プール手数料なし。

### デメリット

- **極端な分散**: 何年もマイニングしても何も得られない可能性があります。
- フィードバックループがない — ブロックをヒットする（かもしれない）まで、セットアップが機能しているかさえわかりません。
- 主要ネットワークで実用的であるためには膨大なハッシュレートが必要です。

### ソロマイニングに向いている人

- 定期的にブロックをヒットできる十分なハッシュレートを持つ大規模運用。
- 宝くじ券として扱い、収益ゼロを気にしない趣味の人。

---

## プールマイニング — 宝くじ組合に参加する

マイナーのグループに参加します。全員が協力し、プール内の **誰か** がブロックを見つけると、各参加者の貢献作業量（**シェア** で測定）に基づいて報酬が分配されます。

ブロックを何世紀も待つ代わりに、少額だが **定期的な支払い** を — 時に毎日、時に毎時間 — 得ることができます。

### シェアの仕組み

プールはネットワーク難易度よりはるかに低い **プール難易度** を設定します。これにより、NMMiner のような小規模マイナーでも数秒から数分ごとに有効な *シェア* を送信できます。これらのシェアは「本当に探索しています」という証明になります。ブロックが見つかると、プールは全員のシェアを集計して支払いを計算します。

### メリット

- **予測可能で定期的な収入** — すぐに結果が見えます。
- **どんなハッシュレート** でも、どんなに小さくても機能します。
- ハードウェアが正しく機能しているというフィードバックが得られます。
- 低リスク — 収益は時間とともに平滑化されます。

### デメリット

- プール手数料（通常収益の 0.5%～3%）。
- プール運営者が公正に支払うと信頼する必要があります。
- ブロックあたりの収益はごくわずかです（しかし積み重なります）。

### プールマイニングに向いている人

- **小～中程度のハッシュレートを持つすべての人** — これにはすべての NMMiner ユーザーが含まれます。
- 一貫した予測可能な収益を望む人。
- 結果を見て学びたい初心者。

## 並べて比較

| | ソロマイニング | プールマイニング |
|---|---|---|
| **感覚** | 宝くじ券 | 定期的な給料 |
| **支払い頻度** | 極めて稀（ほぼない） | 頻繁（毎日 / 毎時間） |
| **1 ヒットあたりの支払い額** | 莫大（ブロック報酬全額） | ごくわずか（ブロックの自分の取り分） |
| **分散** | 極めて高い | 低い |
| **適切なハッシュレート** | TH/s 以上 | どんなハッシュレートでも |
| **手数料** | なし | 0.5%～3% のプール手数料 |
| **NMMiner 対応？** | はい（SoloBTC プール） | はい（標準プール） |

## 本当の疑問：「ソロマイニングなのになぜプールに接続するの？」

ここが多くの初心者が混乱するところです。「プール」という言葉は「ソロ」の反対のように聞こえます。層を剥がして解説しましょう。

### マイニングの 3 つの方法 — 報酬が変わるのは 2 つだけ

すべてのマイナーは、パイプラインのどこかに **Bitcoin ノード** を必要とします — Bitcoin ネットワークと通信し、ブロックチェーンを追跡し、作業を配布するソフトウェアです。問題は：誰がそのノードを運営し、どのような種類の作業を提供するのか？

最も DIY から最も手間なしまで、3 つのセットアップがあります：

| セットアップ | ノード運営者 | ジョブ難易度 | ブロックヒット時の報酬 |
|---|---|---|---|
| **真のソロ**（自前ノード） | あなた | 完全なネットワーク難易度 | 100% あなたのもの |
| **ソロプール**（例：SoloBTC） | プール運営者 | 完全なネットワーク難易度 | 約 98～100%（少額の手数料） |
| **通常プール** | プール運営者 | 低い（シェア難易度） | 全参加者で分割 |

:::info

パターンに注目してください：**「ソロ」は報酬モデルを表し、インフラを誰が運営するかではありません。** 真のソロとソロプールマイニングの両方で、ヒットすればブロックの（ほぼ）全額を取得します。通常のプールでは、報酬は常に分割されます。

:::

### 真のソロマイニング — 「純粋な」バージョン

最も厳密な意味では、ソロマイニングとはあなたのマイナーが **自分のマシンで動作する Bitcoin フルノード** と直接通信することを意味します。ブロックチェーン全体（約 600 GB）をダウンロードし、24 時間 365 日同期を保ち、あなたのマイナーがそれに対して探索します。ブロックをヒットしたら？自分でブロードキャストします。仲介者なし、手数料なし、信頼すべき相手なし。

落とし穴はインフラのオーバーヘッドです。数百 GB の空きディスク容量を持つコンピューター、信頼性の高い常時接続のインターネット回線、ノードを健全に保つための十分な技術的知識が必要です。ほとんどの趣味の人 — 特に小さな ESP32 マイナーを動かしている人 — にとって、これは単純に手間に見合いません。

### ソロプール — ノードの面倒なしのソロ報酬

ここで SoloBTC のようなプールが登場します。彼らはフルノードを運営し、ブロックチェーンの配管作業をすべて **あなたに代わって** 処理します。しかし通常のプールとは異なり、ジョブ難易度を**下げず**、ブロック報酬を参加者間で**分割しません**。

代わりに、接続された各マイナーは **完全な Bitcoin ネットワーク難易度** でジョブを受け取ります。あなたのマイナーがたまたま有効なブロックを見つけた場合、プールがそれをブロードキャストし、報酬を回収し、ほぼ全額をあなたのウォレットに送金します（通常 1～2% の少額の手数料のみを保持）。

重要な洞察：**報酬の観点からは、これはソロマイニングです。** インフラの観点からは、プール支援型です。600 GB のサーバーを運用する必要なしに、ソロの宝くじ券を得られます。

### ではなぜソロは依然として「宝くじ」なのか？

ブロックを見つける難易度は **プールとは無関係** だからです — それは Bitcoin ネットワークによって設定されます。自前のノードを運用するか SoloBTC に接続するかにかかわらず、KH/s 級のハッシュレートは依然として同じ天文学的なオッズに直面します：ヒット間の平均は数世紀です。

プールはインフラの面倒を省いてくれます。数学を変えることはできません。

### NMMiner のデフォルトはどれ？

NMMiner はデフォルトプールとして **SoloBTC** が出荷されます — つまり箱出しでソロプールモードです。これはプラグインした瞬間に「宝くじを遊ぶ」楽しいゼロ設定の方法です。しかし安定した予測可能な収益が欲しい場合は、通常のプールに切り替えてください。

:::tip

**NMMiner ユーザー向け**: Configuration Portal → Pools タブに移動し、デフォルトの SoloBTC URL を従来のマイニングプール URL に置き換えてください。手順の詳細は [プール](../user-guide/pools.md) ページを参照してください。

:::

## どちらを選ぶべきか？

| あなたの目標 | 推奨 |
|---|---|
| 「定期的で予測可能な収益を見たい」 | 通常プール |
| 「いつかフルブロックをヒットするかもしれないスリルを味わいたい」 | ソロプール（例：SoloBTC） |
| 「大規模なマイニングファームを運営し定期的にブロックをヒットしている」 | 真のソロ（自前ノード）またはソロプール |
| 「マイニングの仕組みを学んでいる」 | 通常プール（即時のシェアフィードバック） |

**結論**: 混乱は通常こうです — *「待って、ソロマイニングなのになぜ『プール』に接続するの？」* 答えはこうです：

- **「ソロ」は報酬が共有されないことを意味します。** ブロックを見つけたらそれはあなたのものです — 自前のノードを通じて見つけようと、あなたの代わりにノードを運用するソロプールを通じて見つけようと。
- **「プール」（「ソロプール」における）はインフラ層のみを意味します。** プールが Bitcoin ノードを運用するため、600 GB のハードドライブや 24 時間 365 日のサーバーは不要です。

NMMiner ユーザーにとっての実践的なポイント：**通常プール = 実際に目に見える安定した少額の収益。ソロプール = ほとんどインフラの手間のない宝くじ券。** ソロが「間違い」なのではなく、異なる目的を果たすだけです。
