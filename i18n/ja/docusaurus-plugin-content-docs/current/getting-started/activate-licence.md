# ライセンスのアクティベーション

NMMiner はデバイスごとに **有償ライセンス** が必要です。ライセンスはボードをファームウェアに紐付け、マイニング機能をアンロックします。

## ライセンスの入手方法

1. フラッシュ後、[フラッシュツール](https://flash.nmminer.com) の画面指示に従って **ライセンスを購入** します — ツールがプロセスをガイドします。
2. 購入が完了すると、ツールにライセンス文字列が表示されます。フラッシュツール内またはデバイス上で直接アクティベーションできます。

## アクティベーション方法

ライセンスをアクティベートする方法は 2 つあります：

### 方法 1 — フラッシュツールでアクティベート（推奨）

1. [フラッシュツール](https://flash.nmminer.com) で、ライセンス文字列を **Licence** 入力ボックスに貼り付けます。
2. **Activate** ボタンをクリックします。ツールがライセンスをデバイスに直接書き込みます。
3. 成功すると確認が表示されます — これ以上の手順は不要です。

### 方法 2 — デバイスの設定ポータルでアクティベート

1. ボードが自動的に再起動していない場合は、電源を入れ直します。
2. オープンホットスポット `nmap-2.4g` に接続し、キャプティブポータルを開きます（または `http://192.168.4.1` にアクセスします）。
3. ライセンス文字列を **Licence** ボックスに貼り付け、ボタンを押します。
4. 成功すると、ページに確認が表示され、マイナーが通常モードで再起動します。

:::tip ライセンスはボードに紐付けられます
ライセンスは特定のボードにリンクされています。ファームウェアアップデートでもライセンスは保持されます — アクティベーションはデバイスごとに 1 回のみ必要です。
:::
