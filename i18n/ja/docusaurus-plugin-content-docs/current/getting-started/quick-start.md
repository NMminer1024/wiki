# クイックスタート

**約 10 分** で新品の ESP32 ボードをハッシュ計算開始状態にします。

## 必要なもの

- [公式サポートボード](../hardware/overview.md)（ESP32-D0 / S3 / C3 / C5）。
- USB ケーブルと **Chrome** または **Edge** 搭載の PC（ブラウザベースのフラッシュツール用）。
- 2.4 GHz WiFi ネットワーク。

開始するためにウォレットは **必要ありません** — NMMiner は独自の SoloBTC プール（[solobtc.nmminer.com](https://solobtc.nmminer.com)）と一時的なデモウォレットで事前設定された状態で出荷されます。準備ができたらウォレットを自分の BTC アドレスに置き換えてください。

## 5 つのステップ

1. **ファームウェアをフラッシュ** — [flash.nmminer.com](https://flash.nmminer.com) を開き、ボードを選び、*Install* をクリックします。[ファームウェアのフラッシュ](./flash-firmware.md) を参照。
2. **ライセンスの購入とアクティベーション** — フラッシュツールがライセンス購入をガイドし、ライセンス文字列をアクティベーションボックスに貼り付けます。[ライセンスのアクティベーション](./activate-licence.md) を参照。
3. スマートフォンで **AP `nmap-2.4g`** に接続します。キャプティブポータルが自動的にポップアップします。PC の場合：手動で `http://192.168.4.1` にアクセスします。
4. ウェブページで WiFi（必須）とウォレット（推奨）を **設定** し、**Save** を押します。マイナーは約 15 秒で再起動します。[初回設定](./first-configuration.md) を参照。
5. **マイニングを観察** — 画面の Miner ページにライブハッシュレート、承認済みシェア、プール難易度が表示されます。[solobtc.nmminer.com](https://solobtc.nmminer.com) にアクセスすると、プールダッシュボードに自分のワーカーが表示されます。

:::tip
初回起動時、マイナーは最後に接続した WiFi に 15 秒間再接続を試みます。失敗すると自動的に AP / 設定モードにフォールバックします。
:::
