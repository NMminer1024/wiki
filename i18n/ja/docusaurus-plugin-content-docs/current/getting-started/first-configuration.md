# 初回設定

新しくフラッシュされたボードでは、NMMiner は **AP / 設定モード** に入り、WiFi、プール、ウォレット、ディスプレイオプションを設定するウェブページを公開します。

## フィールド

| フィールド           | デフォルト                                                | 目的                                                              |
| --------------- | ------------------------------------------------------ | -------------------------------------------------------------------- |
| **WiFi SSID**   | `NMTech-2.4G`                                          | お使いの 2.4 GHz ネットワーク名。                                           |
| **WiFi パスワード** | `NMMiner2048`                                        | WPA / WPA2 パスワード。                                                 |
| **プール URL（プライマリ）** | `stratum+tcp://solobtc.nmminer.com:3333`        | NMMiner 独自の SoloBTC プール。本当に必要な場合のみ上書きしてください。     |
| **プール URL（フォールバック）** | `stratum+tcp://au.solobtc.nmminer.com:3333`    | NMMiner のアジア太平洋フォールバック。プライマリが到達不能な場合に使用。 |
| **ウォレット / ワーカー名** | デモウォレットが事前入力済み                        | BTC 支払い先アドレス。オプションで `.workerName` を付加。**変更してください！** |
| **UI リフレッシュ** | ボード固有                                         | 低 → 画面リフレッシュが速く、ハッシュレートはわずかに低下。高 → 最大ハッシュレート、UI は遅くなります。 |
| **画面の明るさ** | 100 %                                           | 0–100 %.                                                              |
| **タイムゾーン**  | `+8`                                                   | 小数対応（例：`8`、`5.5`、`-3.5`）。範囲: -13.5 ~ +13.5。     |
| **時刻形式**| 24 h                                                   | 12 / 24 時間。                                                         |
| **LED**        | ボード固有                                         | ボード上のステータス LED の有効 / 無効。                            |

**Save** を押すと、マイナーはフラッシュへのコミットに 10〜15 秒かかります。**手動でリセットしないでください。** 自動的に再起動します。

:::tip 自分のウォレットを設定してください
NMMiner はボードが「箱から出してすぐに動作する」ようにデモアドレスにマイニングするよう事前設定されています。本格的に使用する前に、ウォレットを自分の BTC アドレスに置き換えてください — そのデモアドレスはすべての新規デバイスで共有されています。
:::

## 再設定の強制

マイナーが既にオンラインの状態で設定を変更する必要がある場合：

- **1 ボタンボード** → `BOOT` を長押し。
- **2 ボタンボード** → `USER` を長押し。
- **タッチスクリーンボード** → 画面を 10 秒間長押し。

マイナーは AP モードで再起動し、キャプティブポータルを再度公開します。詳細な動作：[ボタン](../user-guide/buttons.md)。

## 再起動不要のライブ更新

いくつかの設定は**即座に**適用されます — 再保存や再起動は不要です：

- 画面の明るさ
- 画面の回転
- LED オン / オフ
- タイムゾーン
- スクリーンセーバーモード（image / black）

これらは [NM Monitor](../user-guide/nm-monitor.md) から変更でき、即座に効果を確認できます。
