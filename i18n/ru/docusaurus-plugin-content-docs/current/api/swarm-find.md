# Swarm — Найти меня (Find Me)

Запускает визуальную и светодиодную индикацию на целевом майнере, чтобы вы могли физически его найти. Это API, лежащее в основе кнопки **Find** в меню [Swarm](../user-guide/swarm.md).

---

## `POST /api/swarm/find`

### Запрос

Без тела.

### Ответ

```json
{ "status": "ok" }
```

Целевой майнер:

- Мигает экраном в течение нескольких секунд.
- Мигает LED (если включён).

### Пример — bash

```bash
curl -X POST http://192.168.1.43/api/swarm/find
```

### Пример — Python

```python
import requests

for ip in ["192.168.1.43", "192.168.1.44", "192.168.1.45"]:
    requests.post(f"http://{ip}/api/swarm/find")
    print(f"пинг отправлен на {ip}")
```

:::tip
Объедините с [`GET /alive`](./discovery.md#get-alive), чтобы получить все IP, затем пройдите по ним в цикле — идеально для кнопки «найти все мои майнеры» на вашей домашней панели мониторинга.
:::
