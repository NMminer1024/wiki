# Настройки рынка (Market)

Определяет, что отображается на **странице Price** пользовательского интерфейса устройства.

---

## `GET /api/setting/market`

### Ответ

```json
{
  "MainCoin":      "BTC",
  "WatchCoins":    "BTC,ETH,SOL,DOGE",
  "KlineRotate":   "on",
  "KlineInterval": "1h",
  "PricePageMode": "kline"
}
```

| Поле             | Тип    | Значение                                                          |
| --------------- | ------ | ----------------------------------------------------------------- |
| `MainCoin`      | string | Основная монета, отображаемая в большом ценовом блоке.            |
| `WatchCoins`    | string | Список наблюдения, разделённый запятыми. Не более **20 монет**.   |
| `KlineRotate`   | string | `"on"` для автоматического перебора списка наблюдения.            |
| `KlineInterval` | string | Интервал свечей K-line, например `"15m"`, `"1h"`, `"4h"`, `"1d"`. |
| `PricePageMode` | string | `"kline"` для графика, `"list"` для простого списка цен.          |

---

## `POST /api/setting/market`

Частичное обновление. Вызывает немедленное обновление рынка на следующем тике.

### Запрос

```json
{
  "MainCoin":      "BTC",
  "WatchCoins":    "BTC,ETH,SOL",
  "KlineInterval": "1d"
}
```

### Ответ

`200 OK` с телом `OK`. Монеты сверх лимита в 20 монет молча отбрасываются.

---

## `GET /api/market/pairs`

Возвращает **полный список пар USDT**, которые устройство в данный момент имеет в кэше от источника — полезно для заполнения автодополнения в вашем собственном интерфейсе.

### Ответ

```json
["BTC","ETH","SOL","DOGE","XRP","ADA","..."]
```

Обычный JSON-массив символов базовых монет (суффикс `USDT` удаляется на стороне сервера).

### Пример

```bash
curl http://192.168.1.42/api/market/pairs | jq length
```
