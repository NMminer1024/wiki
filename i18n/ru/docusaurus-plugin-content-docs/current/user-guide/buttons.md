# Кнопки и сенсор

Разные платы имеют одну кнопку (`BOOT`), две (`BOOT` + `USER`) или сенсорный экран. NMMiner предоставляет единообразные жесты для всех трёх вариантов.

## Платы с одной кнопкой

| Жест            | Действие                                            |
| --------------- | --------------------------------------------------- |
| Одинарный клик  | Пробуждение экрана (на платах с дисплеем).          |
| Двойной клик    | Переключение на следующую страницу экрана.          |
| Длинное нажатие | Вход в конфигурацию майнера (принудительный режим AP). |

## Платы с двумя кнопками

| Кнопка  | Жест            | Действие                                                  |
| ------- | --------------- | --------------------------------------------------------- |
| `BOOT`  | Одинарный клик  | Пробуждение экрана.                                       |
| `BOOT`  | Двойной клик    | Пробуждение экрана.                                       |
| `BOOT`  | Длинное нажатие | Очистка всего сохранённого состояния из NVS (если включено в сборке). |
| `USER`  | Одинарный клик  | Пробуждение экрана.                                       |
| `USER`  | Двойной клик    | Переключение на следующую страницу экрана.                |
| `USER`  | Длинное нажатие | Вход в конфигурацию майнера (принудительный режим AP).    |

## Платы с сенсорным экраном

Начиная с v2.0.01, сенсорные платы (NM-TV-154, CYD 2.4 / 2.8 / 3.5, Waveshare 3.5 touch и т.д.) поддерживают жесты **непосредственно на экране**:

| Жест                            | Действие                                            |
| ------------------------------- | --------------------------------------------------- |
| Касание (Tap)                   | Переключение на следующую страницу экрана (также пробуждает экран, если он в спящем режиме). |
| **Длинное нажатие 10 секунд**   | Вход в конфигурацию майнера (принудительный режим AP). |

См. [Экраны](./screens.md) для порядка страниц и [Конфигурационный портал](./configuration-portal.md) для того, что происходит после длинного нажатия.
