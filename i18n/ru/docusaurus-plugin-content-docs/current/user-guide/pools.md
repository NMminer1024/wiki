# Пулы

Поскольку хэшрейт ESP32 невелик (несколько сотен KH/s), NMMiner подключается к **пулам с низкой сложностью**, которые принимают шары значительно ниже сложности сети.

NMMiner поставляется с предварительной настройкой на наш собственный пул — **обычно вам ничего не нужно менять**.

## Встроенные значения по умолчанию

| Слот       | URL                                              | Регион        |
| ---------- | ------------------------------------------------ | ------------- |
| Основной   | `stratum+tcp://solobtc.nmminer.com:3333`         | Глобальный    |
| Резервный  | `stratum+tcp://au.solobtc.nmminer.com:3333`      | Азия-Тихоокеанский |

Оба являются официальными пулами NMMiner [SoloBTC](https://solobtc.nmminer.com), разработанными для майнеров с сотнями KH/s. Панель: [solobtc.nmminer.com](https://solobtc.nmminer.com).

Когда основной пул становится недоступен, майнер автоматически переключается на резервный. Когда основной восстанавливается, майнер переключается обратно в течение ~10 секунд (начиная с v1.7.05).

## Поддерживаемые схемы URL пулов

| Схема                 | Значение                                               |
| -------------------- | ----------------------------------------------------- |
| `stratum+tcp://`     | Обычный TCP Stratum v1.                               |
| `stratum+ssl://`     | TLS-обёрнутый Stratum (используйте, если ваш пул требует SSL). |

## Поддерживаемые монеты

NMMiner в настоящее время поддерживает **Bitcoin (BTC)** и **DigiByte (DGB)**. Оба используют SHA-256d, поэтому один и тот же конвейер хеширования обслуживает оба. См. [Список пулов](../reference/pool-list.md) для проверенных конечных точек для каждого.

## Почему не крупные коммерческие пулы?

Хэшрейт NMMiner составляет порядка сотен KH/s. Пулы, которые требуют высокой минимальной сложности шары, будут отклонять каждую шару NMMiner. Придерживайтесь официальных пулов NMMiner [SoloBTC](https://solobtc.nmminer.com) или пулов сообщества соло-типа — всего, что принимает очень низкую сложность шары.
