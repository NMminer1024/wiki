# NM Monitor

**NM Monitor** — это веб-приложение, которое живёт **внутри каждого устройства NMMiner**. Ничего не нужно устанавливать на телефон или ПК — откройте IP майнера в любом браузере, и вы внутри. Весь мониторинг и управление парком устройств осуществляется через NM Monitor и его меню [Swarm](./swarm.md).

## Открытие NM Monitor

1. Подключите телефон или ПК к **той же WiFi**, что и майнер.
2. Найдите IP майнера — проще всего проверить **страницу Miner** на экране устройства (IP отображается в строке состояния). Или найдите его в таблице DHCP вашего роутера.
3. Откройте `http://<miner-ip>/` в **Chrome, Edge, Safari или Firefox**. Порт запоминать не нужно — порт 80.
4. Домашняя страница загружается автоматически.

:::tip
Добавьте в закладки `http://<miner-hostname>/` (например, `http://nm-cyd-c5-ab12cd/`) — большинство домашних роутеров также позволяют обращаться к майнеру по имени хоста.
:::

## Что можно делать в NM Monitor

| Раздел               | Назначение                                                          |
| -------------------- | ------------------------------------------------------------------- |
| **Dashboard**        | Текущий хэшрейт, принятые / отклонённые шары, сложность пула, время работы, RSSI, свободная куча, температура (где поддерживается). |
| **Swarm**            | Агрегация по всей LAN: просмотр всех NMMiner в одной WiFi, суммарный хэшрейт, пинг конкретного майнера для его мигания. См. [Swarm](./swarm.md). |
| **Network**          | Изменение имени хоста, SSID WiFi, пароля WiFi.                      |
| **Mining**           | URL основного / резервного пула, адрес кошелька, пароль пула.       |
| **Time**             | Часовой пояс, формат времени (12 / 24 ч), формат даты.              |
| **Preferences**      | Яркость экрана, поворот, LED вкл/выкл, таймаут заставки, режим заставки (image / black), согласие на [Ladder](./ladder.md). |
| **Market**           | Выбор основной монеты и списка наблюдения для страницы Price.       |
| **Weather**          | Настройка города / широты / долготы и единиц измерения для страницы Weather. |
| **Screensaver**      | Загрузка пользовательских PNG, которые конвертируются в браузере и отправляются на майнер. |
| **System**           | Версия прошивки, плата, свободная куча, кнопка **Restart**.          |
| **API Doc**          | Живой справочник HTTP API (также опубликован по `/api-doc`).         |

## Чтение из любого места в вашей LAN

NM Monitor устанавливает `Access-Control-Allow-Origin: *` на каждой конечной точке API. Это означает, что браузерная страница с любого источника может читать статус майнера — полезно для создания собственных панелей мониторинга. Полный набор конечных точек документирован в [Справочнике API](../api/overview.md).
