# Обновление прошивки

Обновление такое же, как и первая прошивка: снова откройте браузерный инструмент прошивки, выберите плату, установите.

## Шаги

1. Откройте [flash.nmminer.com](https://flash.nmminer.com) в Chrome или Edge.
2. Выберите профиль платы.
3. Нажмите **Install**, выберите последовательный порт, дождитесь завершения.

После обновления вам **не нужно повторно активировать лицензию**. Настройки (SSID / пул / кошелёк) сохраняются при обновлении.

## Удалённое отслеживание версии прошивки

Текущая версия прошивки сообщается каждым майнером через [`GET /api/system/info`](../api/system.md). Объедините это с обнаружением [Swarm](./swarm.md), и вы сможете с одного взгляда увидеть, какие майнеры в LAN ещё нуждаются в обновлении.

## Что меняется между версиями

См. [журнал релизов GitHub](https://github.com/NMminer1024/NMMiner#release-log) для полного списка изменений. Мажорные версии (например, **v1.x → v2.0**) вносят критические улучшения в NM Monitor и Swarm — старые релизы несовместимы с новыми протоколами Swarm, поэтому лучше держать весь парк на одной минорной серии.
