# Заставка (Screen Saver)

После настраиваемого периода бездействия дисплей переходит в режим заставки.

## Режимы

| Режим   | Эффект                                                                                  |
| ------- | --------------------------------------------------------------------------------------- |
| `IMAGE` | Слайд-шоу из пользовательских изображений, загруженных через NM Monitor. Плавное появление / исчезновение. |
| `BLACK` | Выключает подсветку (OLED переходят на минимальную контрастность). Экономит энергию и **даёт абсолютный пиковый хэшрейт**. |

Переключайте режимы на лету из раздела **Preferences** в [NM Monitor](./nm-monitor.md) — без перезагрузки.

## Пользовательские изображения

Страница **Screensaver** в NM Monitor позволяет загружать обычные файлы PNG / JPG прямо из браузера. Браузер преобразует каждое изображение в компактный формат `.ss` NMMiner (RGB565 + RLE) и отправляет его в папку `/ss/` устройства.

- Изображения автоматически подгоняются под **текущее разрешение экрана**.
- Каждый файл `.ss` не превышает **200 КБ**.
- Устройство управляет дисковым пространством — если ФС заполняется, самая старая заставка перезаписывается.

Хотите автоматизировать загрузку? См. [API › Загрузка заставки](../api/screensaver-upload.md).
