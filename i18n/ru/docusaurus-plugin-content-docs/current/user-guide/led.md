# Светодиодный индикатор (LED)

Платы со встроенным RGB или статусным LED используют его как индикатор сердцебиения майнера в реальном времени.

| Состояние           | Шаблон LED                               |
| ------------------- | ---------------------------------------- |
| Подключение к WiFi  | Дыхание (Breathing)                      |
| Майнинг             | Медленная пульсация, синхронизированная с отправкой шар |
| Шара принята        | Быстрая яркая вспышка                    |
| Шара отклонена      | Быстрая тусклая вспышка / выключение     |
| Ошибка              | Постоянный цвет (зависит от модели)      |

LED можно **отключить** в конфигурационном портале, если свет вас беспокоит.
