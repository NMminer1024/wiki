# Страницы экрана

Платы с дисплеем циклически переключаются между небольшим набором страниц. Переключение осуществляется жестом, описанным в [Кнопки и сенсор](./buttons.md) — касание, двойной клик или свайп в зависимости от оборудования.

NMMiner поставляется с макетами для следующих нативных размеров:

```
128x64    128x128    160x80     240x135
240x240   320x170    320x172    320x240
480x320   536x240
```

## Страницы

| Порядок | Страница     | Назначение                                                                |
| ------- | ------------ | ------------------------------------------------------------------------ |
| 1       | **Loading**  | Заставка + статус загрузки, версия прошивки, сетевое рукопожатие.         |
| 2       | **Config**   | Отображается в режиме AP — QR / IP / SSID для доступа к каптивному порталу. |
| 3       | **Miner**    | Текущий хэшрейт, лучшая сложность за **сессию** и **за всё время**, принятые / отклонённые шары, URL пула, время работы. Основная «рабочая» страница. |
| 4       | **Clock**    | Большие часы + дата, с тикером цены BTC, который зеленеет при росте и краснеет при падении. Выбираемый формат времени 12 / 24 ч, настраиваемый формат даты. |
| 5       | **Price**    | Два режима:<br/>• **K-line** — свечной график основной монеты, настраиваемый интервал (`15m`, `1h`, `4h`, `1d`, …).<br/>• **Wall** — просмотр списка наблюдения нескольких монет (до 20 пар). |
| 6       | **Weather**  | Текущие условия + AQI + прогноз на 3 дня + восход / закат. Выбираемые единицы температуры (°C / °F), скорости ветра (км/ч / миль/ч / м/с) и альтиметра (давление / влажность). |
| 7       | **Ladder**   | **Опциональная** глобальная таблица топ-10 лидеров, ранжированная по лучшей сложности шары за всё время. Кошельки отображаются как `первые4…последние4` для конфиденциальности. По умолчанию выключена. Те же данные также публикуются в интернете на [ladder.nmminer.com](https://ladder.nmminer.com/) — см. [Ladder](./ladder.md). |

:::info
Порядок страниц одинаков на всех платах; маленькие / OLED экраны пропускают страницы, которые не помещаются. NMMiner запоминает страницу, на которой вы были, и возвращается к ней после цикла питания (начиная с v1.8.20).
:::

## Заставка

Если майнер простаивает достаточно долго, он входит в режим заставки:

- **Режим Image** — слайд-шоу из пользовательских изображений, загруженных через NM Monitor.
- **Режим Black** — выключает подсветку (или контрастность OLED до минимума) для минимального энергопотребления и **абсолютного пикового хэшрейта**.

Жесты пробуждения: одинарный клик `BOOT` (или `USER`) или касание сенсорного экрана. Подробнее: [Заставка](./screen-saver.md).
