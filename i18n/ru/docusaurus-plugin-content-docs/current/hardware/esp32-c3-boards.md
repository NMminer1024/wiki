# Платы ESP32-C3

Одно ядро RISC-V, самый маленький и дешёвый уровень. Пиковый хэшрейт **~402 KH/s**.

| Плата                       | Хэшрейт       | Чип      |
| --------------------------- | -------------- | --------- |
| Seeed XIAO ESP32-C3         | **396.9 KH/s** | ESP32-C3  |
| ESP32-C3 0.42" OLED         | **315.1 KH/s** | ESP32-C3  |
