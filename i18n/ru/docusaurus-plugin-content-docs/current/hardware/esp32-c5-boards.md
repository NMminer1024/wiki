# Платы ESP32-C5

ESP32-C5 — это более новый двухдиапазонный чип RISC-V. NMMiner предоставляет поддержку, включая специализированную отладочную плату NM-CYD-C5.

| Плата                  | Хэшрейт       | Чип      |
| ---------------------- | -------------- | --------- |
| ESP32-C5 DevKitC-1     | **150.2 KH/s** | ESP32-C5  |
| NM-CYD-C5              | **150.2 KH/s** | ESP32-C5  |

:::info
Поддержка C5 продолжает получать улучшения хэшрейта. См. [журнал релизов GitHub](https://github.com/NMminer1024/NMMiner#release-log) для получения последних данных.
:::
