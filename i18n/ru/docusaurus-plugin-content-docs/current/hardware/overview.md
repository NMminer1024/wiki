# Обзор оборудования

NMMiner ориентирован на четыре семейства чипов. Каждое семейство имеет свой пиковый хэшрейт и подобранный список поддерживаемых плат.

| Семейство чипов | Типичный пиковый хэшрейт | Примечания |
| --------------- | ----------------------- | ---------- |
| **ESP32-D0**    | до **1 035 KH/s**       | Лучшая чистая производительность. Оригинальное семейство ESP32. |
| **ESP32-S3**    | до **400 KH/s**         | Векторная ISA, два ядра, обычно в паре с TFT / AMOLED дисплеями. |
| **ESP32-C3**    | до **402 KH/s**         | Одно ядро RISC-V, самый дешёвый уровень. |
| **ESP32-C5**    | до **150 KH/s**         | Более новый двухдиапазонный чип RISC-V. |

Используйте подстраницы, чтобы найти именно вашу плату:

- [Платы ESP32-D0](./esp32-d0-boards.md)
- [Платы ESP32-S3](./esp32-s3-boards.md)
- [Платы ESP32-C3](./esp32-c3-boards.md)
- [Платы ESP32-C5](./esp32-c5-boards.md)

:::tip
Столбец хэшрейта **измерен на реальном оборудовании** при наивысшей стабильной настройке обновления UI, с заставкой в режиме **Black**. Снижение интервала обновления экрана даёт дополнительные несколько KH/s на каждой плате.
:::
