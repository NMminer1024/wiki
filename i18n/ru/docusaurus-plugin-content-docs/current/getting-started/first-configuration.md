# Первая настройка

На свежепрошитой плате NMMiner входит в **режим AP / конфигурации** и предоставляет веб-страницу, где вы задаёте WiFi, пул, кошелёк и параметры дисплея.

## Поля

| Поле           | По умолчанию                                           | Назначение                                                              |
| -------------- | ------------------------------------------------------ | ----------------------------------------------------------------------- |
| **WiFi SSID**  | `NMTech-2.4G`                                          | Имя вашей сети 2.4 ГГц.                                                 |
| **WiFi Password** | `NMMiner2048`                                       | Пароль WPA / WPA2.                                                      |
| **Pool URL (основной)** | `stratum+tcp://solobtc.nmminer.com:3333`        | Собственный пул SoloBTC от NMMiner. Переопределяйте только при реальной необходимости. |
| **Pool URL (резервный)** | `stratum+tcp://au.solobtc.nmminer.com:3333`    | Азиатско-тихоокеанский резерв NMMiner. Используется, если основной пул недоступен. |
| **Кошелёк / Имя воркера** | Демо-кошелёк предзаполнен                  | Ваш BTC-адрес для выплат, опционально с `.workerName`. **Измените это!** |
| **UI refresh** | Зависит от платы                                       | Ниже = быстрее обновление экрана, немного меньше хэшрейт. Выше = макс. хэшрейт, медленнее UI. |
| **Яркость экрана** | 100 %                                            | 0–100 %.                                                                |
| **Часовой пояс** | `+8`                                                | Допускаются дробные значения (например, `8`, `5.5`, `-3.5`). Диапазон: -13.5 ~ +13.5. |
| **Формат времени** | 24 ч                                              | 12 / 24 часа.                                                           |
| **LED**        | Зависит от платы                                       | Включить / отключить статусный LED на плате.                            |

При нажатии **Save** майнеру требуется 10-15 с для сохранения во flash. **Не сбрасывайте вручную.** Он перезапустится самостоятельно.

:::tip Укажите свой собственный кошелёк
NMMiner предварительно настроен на майнинг на демо-адрес, чтобы плата «просто работала» из коробки. Замените кошелёк на свой собственный BTC-адрес перед тем, как делать что-либо серьёзное — этот демо-адрес используется всеми новыми устройствами.
:::

## Принудительная повторная настройка

Если вам нужно изменить настройки после того, как майнер уже в сети:

- **Плата с одной кнопкой** → длительное нажатие `BOOT`.
- **Плата с двумя кнопками** → длительное нажатие `USER`.
- **Плата с сенсорным экраном** → длительное нажатие на экран в течение 10 секунд.

Майнер перезагружается в режим AP и снова предоставляет каптивный портал. Подробное поведение: [Кнопки](../user-guide/buttons.md).

## Изменения на лету без перезагрузки

Несколько настроек применяются **мгновенно** — без необходимости повторного сохранения и перезагрузки:

- Яркость экрана
- Поворот экрана
- LED вкл / выкл
- Часовой пояс
- Режим заставки (image / black)

Вы можете изменить их из [NM Monitor](../user-guide/nm-monitor.md) и сразу увидеть эффект.
