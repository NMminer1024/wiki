# Прошивка

NMMiner поставляется в виде единого образа, прошиваемого через браузер. Не требуется `esptool` или `idf.py`.

## Прошивка через браузер (рекомендуется)

1. Откройте [flash.nmminer.com](https://flash.nmminer.com) в **Chrome или Edge**. Firefox / Safari не реализуют WebSerial.
2. Подключите плату через USB.
3. Выберите свою плату из списка. Инструмент загрузит соответствующую прошивку из официального релиза GitHub.
4. Нажмите **Install**, выберите последовательный порт и дождитесь сообщения "Flash complete".
5. Инструмент предложит вам **приобрести лицензию**. После покупки он отобразит вашу лицензионную строку — скопируйте её. Она понадобится один раз на странице активации.

:::warning
Если ваша плата не входит в режим загрузки автоматически, удерживайте `BOOT` при нажатии `RESET`, затем отпустите.
:::
