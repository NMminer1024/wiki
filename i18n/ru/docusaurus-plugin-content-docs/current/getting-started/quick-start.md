# Быстрый старт

Запустите новую плату ESP32 в хеширование примерно за **10 минут**.

## Что вам понадобится

- [Официально поддерживаемая плата](../hardware/overview.md) (ESP32-D0 / S3 / C3 / C5).
- USB-кабель и ПК с **Chrome** или **Edge** (для браузерного инструмента прошивки).
- Сеть WiFi 2.4 ГГц.

Вам **не нужен кошелёк** для начала — NMMiner поставляется с предварительной настройкой на наш собственный пул SoloBTC ([solobtc.nmminer.com](https://solobtc.nmminer.com)) и временный демо-кошелёк. Замените кошелёк на свой собственный BTC-адрес, когда будете готовы.

## Пять шагов

1. **Прошейте прошивку** — откройте [flash.nmminer.com](https://flash.nmminer.com), выберите свою плату, нажмите *Install*. См. [Прошивка](./flash-firmware.md).
2. **Купите и активируйте лицензию** — инструмент прошивки проведёт вас через покупку лицензии, затем вставьте лицензионную строку в поле активации. См. [Активация лицензии](./activate-licence.md).
3. **Подключитесь к AP `nmap-2.4g`** с телефона. Каптивный портал появится автоматически. ПК: посетите `http://192.168.4.1` вручную.
4. **Настройте** WiFi (обязательно) и кошелёк (рекомендуется) на веб-странице, затем нажмите **Save**. Майнер перезапустится через ~15 секунд. См. [Первая настройка](./first-configuration.md).
5. **Наблюдайте за майнингом** — страница Miner на экране показывает текущий хэшрейт, принятые шары и сложность пула. Посетите [solobtc.nmminer.com](https://solobtc.nmminer.com), чтобы увидеть своего воркера на панели пула.

:::tip
При первом включении майнер попытается переподключиться к последней известной WiFi в течение 15 секунд. Если не удастся, он автоматически возвращается в режим AP / конфигурации.
:::
