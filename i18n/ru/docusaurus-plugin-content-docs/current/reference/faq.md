# Часто задаваемые вопросы (FAQ)

## Найдёт ли NMMiner когда-нибудь блок Bitcoin самостоятельно?

Математически — да, с той же вероятностью, что и лотерейный билет. Практически — нет. Смысл в **удовольствии** (и поддержке сети), а не в выплате.

## Сколько электричества он потребляет?

Большинство поддерживаемых плат потребляют менее 1 Вт. Даже при $0.30 /кВт·ч это значительно меньше 30 центов в месяц.

## Какой пул NMMiner использует по умолчанию?

NMMiner поставляется с предварительной настройкой на собственный пул SoloBTC:

- Основной: `stratum+tcp://solobtc.nmminer.com:3333`
- Резервный: `stratum+tcp://au.solobtc.nmminer.com:3333`

Вам **нужно только изменить адрес кошелька** на свой собственный BTC-адрес. URL пулов обычно можно оставить как есть.

## Могу ли я использовать другой пул?

Да — всё, что говорит на Stratum v1, будет работать. См. [Список пулов](./pool-list.md) для вариантов сообщества SoloBTC и DGB. Обратите внимание, что крупные коммерческие пулы с высокой минимальной сложностью шары будут отклонять каждую шару — придерживайтесь официальных пулов NMMiner [SoloBTC](https://solobtc.nmminer.com) или конечных точек соло-типа.

## Может ли NMMiner майнить что-то кроме Bitcoin?

Да — **DigiByte (DGB)** также поддерживается. DGB использует тот же алгоритм SHA-256d, поэтому он работает на той же прошивке. Просто укажите URL пула на пул DGB и используйте адрес кошелька DGB.

## Нужен ли мне кошелёк на устройстве?

Нет. Вам нужен только **адрес для выплат**. Пул отправляет награды напрямую на любой BTC / DGB адрес, который вы настроите.

## Мой экран просыпается сам по себе каждые несколько секунд

Это безвредное фоновое обновление. Используйте режим заставки **Black**, если хотите нулевой активности дисплея (и максимальный хэшрейт).

## Могу ли я запустить два майнера в одной WiFi?

Да. Меню **Swarm** в NM Monitor автоматически обнаруживает все NMMiner в одной LAN — до 255 устройств на подсеть /24. См. [Swarm](../user-guide/swarm.md).

## Мой майнер завис на экране Loading / никогда не подключается к пулу

Пройдите по этому контрольному списку:

1. **WiFi должен быть 2.4 ГГц.** Большинство чипов ESP32 не поддерживают 5 ГГц.
2. **Проверьте адрес кошелька.** Корректный Bitcoin-адрес начинается с `bc1q...`, `bc1p...`, `1...` или `3...`.
3. **Проверьте URL пула.** Из коробки NMMiner предварительно настроен с `stratum+tcp://solobtc.nmminer.com:3333`.
4. **Сложность пула.** Крупные коммерческие пулы с высокой минимальной сложностью шары будут отклонять каждую шару NMMiner.
5. **Источник питания.** Нестабильный USB-порт может вызывать просадки напряжения. Попробуйте другой кабель + источник питания 5 В/1 А.

## Есть ли публичное API?

Да — каждое устройство предоставляет документированное HTTP API на порту 80. См. [Справочник API](../api/overview.md).

## Доступен ли исходный код?

Прошивка NMMiner является закрытой. HTTP API на каждом устройстве — это поддерживаемая поверхность для интеграции. Сопутствующий CLI-инструмент прошивки и веб-интерфейсы пула SoloBTC являются публичными.

## Что делает страница Ladder?

Это **опциональная** глобальная таблица топ-10 лидеров, ранжированная по лучшей сложности шары за всё время, полученной участвующими майнерами. Кошельки отображаются в формате `первые4…последние4` для конфиденциальности. По умолчанию отключена — включите её в разделе **Preferences** NM Monitor. Публичный веб-вид находится на [ladder.nmminer.com](https://ladder.nmminer.com/) — см. [Ladder](../user-guide/ladder.md) для полной информации.
