# Глоссарий

| Термин                   | Определение                                                                       |
| ----------------------- | -------------------------------------------------------------------------------- |
| **AP mode**             | Каптивный портал Soft-AP, в который NMMiner переходит, когда не может подключиться к WiFi. |
| **CYD**                 | "Cheap Yellow Display" — семейство недорогих плат ESP32 + TFT.                   |
| **DGB / DigiByte**      | Альткойн на SHA-256d. NMMiner поддерживает DGB из коробки — та же прошивка, другой пул. |
| **Hashrate**            | Хешей в секунду. В NMMiner отображается в KH/s.                                  |
| **Job**                 | Единица работы, выдаваемая пулом (`mining.notify`).                              |
| **Ladder**              | Опциональная глобальная таблица топ-10 лидеров по сложности, публикуемая на [ladder.nmminer.com](https://ladder.nmminer.com/) и зеркалируемая на экране устройства. |
| **Licence**             | Строка активации на устройство, привязывающая плату к образу прошивки.           |
| **NM Monitor**          | Браузерный интерфейс, обслуживаемый каждым майнером на порту 80.                 |
| **Nonce**               | 32-битное поле, которое майнер перебирает при поиске шары.                       |
| **Pool diff**           | Минимальная сложность шары, принимаемая пулом.                                   |
| **Probe**               | Лёгкая конечная точка `/probe`, используемая для идентификации в LAN.            |
| **Share**               | Хеш, удовлетворяющий сложности пула, но не сложности сети.                       |
| **SoloBTC**             | Официальный майнинг-пул NMMiner: [solobtc.nmminer.com](https://solobtc.nmminer.com). |
| **Stratum**             | Протокол пула, на котором работает NMMiner. Только v1.                           |
| **Swarm**               | Меню Swarm в NM Monitor — агрегация майнеров по всей LAN, без установки. До 255 устройств на подсеть /24. |
