# Список пулов

Пулы, которые хорошо работают с хэшрейтом NMMiner ~150–1000 KH/s. Все они динамически подстраивают сложность шары для маленьких майнеров.

## Bitcoin (BTC) — **по умолчанию**

| Пул                               | Stratum URL                                       | Панель                                                   |
| --------------------------------- | ------------------------------------------------- | -------------------------------------------------------- |
| **NMMiner SoloBTC** (по умолчанию)| `stratum+tcp://solobtc.nmminer.com:3333`          | [solobtc.nmminer.com](https://solobtc.nmminer.com)       |
| **NMMiner SoloBTC AU** (резервный)| `stratum+tcp://au.solobtc.nmminer.com:3333`       | [au.solobtc.nmminer.com](https://au.solobtc.nmminer.com) |

Пулы SoloBTC управляются NMMiner. Они предварительно настроены на каждом устройстве — из коробки вам нужно только указать свой адрес кошелька.

## DigiByte (DGB)

| Пул                 | Stratum URL                                       | Панель                                               |
| ------------------- | ------------------------------------------------- | ---------------------------------------------------- |
| Solo Miner DGB      | `stratum+tcp://dgb-stratum.solominer.net:3333`    | [digibyte.solominer.net](https://digibyte.solominer.net) |

DGB использует тот же алгоритм хеширования SHA-256d, что и BTC, поэтому та же прошивка NMMiner работает без каких-либо изменений сборки — различаются только URL пула и адрес кошелька.

## Примечания

:::tip
Если пул поддерживает TLS, переключите схему на `stratum+ssl://` — тот же хост и порт.
:::

:::warning
Крупные коммерческие пулы с высокой минимальной сложностью шары (например, `0x100000000`) будут отклонять каждую шару NMMiner. Придерживайтесь конечных точек типа SoloBTC, которые снижают сложность шары до очень низких значений для маленьких майнеров.
:::
