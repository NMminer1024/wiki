# Activate Licence

NMMiner는 기기당 **유료 라이선스**가 필요합니다. 라이선스는 보드를 펌웨어에 연결하고 마이닝 기능을 활성화합니다.

## 라이선스 받는 방법

1. 플래시 후, [플래시 도구](https://flash.nmminer.com)의 화면 안내에 따라 **라이선스를 구매**하십시오 — 도구가 과정을 안내합니다.
2. 구매가 완료되면 도구에 라이선스 문자열이 표시됩니다. 플래시 도구에서 직접 활성화하거나 기기 자체에서 활성화할 수 있습니다.

## 활성화 방법

라이선스를 활성화하는 두 가지 방법이 있습니다:

### 방법 1 — 플래시 도구에서 활성화 (권장)

1. [플래시 도구](https://flash.nmminer.com)에서 라이선스 문자열을 **Licence** 입력 상자에 붙여넣습니다.
2. **Activate** 버튼을 클릭합니다. 도구가 라이선스를 기기에 직접 기록합니다.
3. 성공하면 확인 메시지가 표시됩니다 — 추가 단계가 필요하지 않습니다.

### 방법 2 — 기기의 구성 포털을 통해 활성화

1. 보드가 자동으로 재부팅되지 않은 경우 전원을 껐다 켭니다.
2. 개방형 핫스팟 `nmap-2.4g`에 연결하고 캡티브 포털이 열리도록 합니다 (또는 `http://192.168.4.1` 방문).
3. 라이선스 문자열을 **Licence** 상자에 붙여넣고 버튼을 누릅니다.
4. 성공하면 페이지에 확인 메시지가 표시되고 마이너가 정상 모드로 재부팅됩니다.

:::tip 라이선스는 보드에 귀속됩니다
라이선스는 특정 보드에 연결됩니다. 펌웨어 업데이트 시 라이선스가 유지됩니다 — 기기당 한 번만 활성화하면 됩니다.
:::
