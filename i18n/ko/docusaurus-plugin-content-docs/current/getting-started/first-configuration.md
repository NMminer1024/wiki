# First Configuration

새로 플래시된 보드에서 NMMiner는 **AP / 구성 모드**로 진입하여 WiFi, 풀, 지갑 및 디스플레이 옵션을 설정할 수 있는 웹 페이지를 노출합니다.

## 필드

| 필드                    | 기본값                                                  | 목적                                                                  |
| ----------------------- | ------------------------------------------------------- | --------------------------------------------------------------------- |
| **WiFi SSID**           | `NMTech-2.4G`                                           | 2.4 GHz 네트워크 이름.                                                |
| **WiFi Password**       | `NMMiner2048`                                           | WPA / WPA2 비밀번호.                                                  |
| **Pool URL (기본)**     | `stratum+tcp://solobtc.nmminer.com:3333`                | NMMiner 자체 SoloBTC 풀. 정말 필요한 경우에만 변경하십시오.           |
| **Pool URL (대체)**     | `stratum+tcp://au.solobtc.nmminer.com:3333`             | NMMiner 아시아-태평양 대체 풀. 기본 풀에 연결할 수 없을 때 사용됩니다. |
| **Wallet / Worker 이름**| 데모 지갑이 미리 채워져 있음                            | BTC 지급 주소, 선택적으로 `.workerName`을 붙임. **반드시 변경하십시오!** |
| **UI refresh**          | 보드별 상이                                              | 낮을수록 화면 새로고침이 빠르고 해시레이트는 약간 감소. 높을수록 최대 해시레이트, UI는 느려짐. |
| **Screen brightness**   | 100 %                                                   | 0–100 %.                                                              |
| **Time zone**           | `+8`                                                    | 소수 지원 (예: `8`, `5.5`, `-3.5`). 범위: -13.5 ~ +13.5.             |
| **Time format**         | 24 h                                                    | 12 / 24시간.                                                           |
| **LED**                 | 보드별 상이                                              | 온보드 상태 LED 활성화 / 비활성화.                                     |

**Save**를 누르면 마이너가 플래시에 커밋하는 데 10-15초가 소요됩니다. **수동으로 재설정하지 마십시오.** 자체적으로 재시작됩니다.

:::tip 자신의 지갑을 설정하십시오
NMMiner는 보드가 "바로 작동"하도록 데모 주소로 마이닝하도록 사전 구성되어 있습니다. 본격적으로 사용하기 전에 지갑을 자신의 BTC 주소로 교체하십시오 — 해당 데모 주소는 모든 새 기기에서 공유됩니다.
:::

## 강제 재구성

마이너가 이미 온라인 상태인 후 설정을 변경해야 하는 경우:

- **단일 버튼 보드** → `BOOT` 길게 누르기.
- **이중 버튼 보드** → `USER` 길게 누르기.
- **터치 스크린 보드** → 화면을 10초간 길게 누르기.

마이너가 AP 모드로 재부팅되고 캡티브 포털을 다시 노출합니다. 자세한 동작: [Buttons](../user-guide/buttons.md).

## 재부팅 없는 실시간 업데이트

일부 설정은 **즉시** 적용됩니다 — 다시 저장하고 재부팅할 필요가 없습니다:

- 화면 밝기
- 화면 회전
- LED 켜기 / 끄기
- 시간대
- 화면 보호기 모드 (이미지 / 블랙)

[NM Monitor](../user-guide/nm-monitor.md)에서 변경하면 즉시 효과를 확인할 수 있습니다.
