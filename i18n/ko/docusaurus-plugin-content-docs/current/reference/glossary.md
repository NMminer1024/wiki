# Glossary

| 용어                    | 정의                                                                                 |
| ----------------------- | ------------------------------------------------------------------------------------ |
| **AP mode**             | NMMiner가 WiFi에 연결할 수 없을 때 전환되는 Soft-AP 캡티브 포털.                    |
| **CYD**                 | "Cheap Yellow Display" — 저가형 ESP32 + TFT 보드 제품군.                             |
| **DGB / DigiByte**      | SHA-256d 알트코인. NMMiner는 기본적으로 DGB를 지원합니다 — 동일 펌웨어, 다른 풀.     |
| **Hashrate**            | 초당 해시 수. NMMiner에서는 KH/s로 보고됩니다.                                       |
| **Job**                 | 풀이 전달하는 작업 단위 (`mining.notify`).                                            |
| **Ladder**              | [ladder.nmminer.com](https://ladder.nmminer.com/)에 게시되고 기기 화면에 미러링되는 옵트인 글로벌 상위 10위 난이도 리더보드. [Ladder](../user-guide/ladder.md) 참조. |
| **Licence**             | 보드를 펌웨어 이미지에 연결하는 기기별 활성화 문자열.                                |
| **NM Monitor**          | 모든 마이너에서 포트 80으로 제공되는 브라우저 기반 UI.                                |
| **Nonce**               | 마이너가 셰어를 검색할 때 반복하는 32비트 필드.                                      |
| **Pool diff**           | 풀이 수락하는 최소 셰어 난이도.                                                      |
| **Probe**               | LAN 식별에 사용되는 경량 `/probe` 엔드포인트.                                        |
| **Share**               | 풀 난이도는 충족하지만 네트워크 난이도는 충족하지 않는 해시.                         |
| **SoloBTC**             | NMMiner의 공식 마이닝 풀: [solobtc.nmminer.com](https://solobtc.nmminer.com). 모든 출하 기기의 기본 풀. |
| **Stratum**             | NMMiner가 사용하는 풀 프로토콜. v1 전용.                                             |
| **Swarm**               | NM Monitor의 Swarm 메뉴 — LAN 전체 마이너 집계, 설치 불필요. /24 서브넷당 최대 255대. |
