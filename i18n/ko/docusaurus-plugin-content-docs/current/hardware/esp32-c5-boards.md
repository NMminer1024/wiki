# ESP32-C5 Boards

ESP32-C5는 최신 듀얼 밴드 RISC-V 칩입니다. NMMiner는 전용 NM-CYD-C5 개발 보드를 포함한 지원을 제공합니다.

| Board                  | Hashrate       | Chip      |
| ---------------------- | -------------- | --------- |
| ESP32-C5 DevKitC-1     | **150.2 KH/s** | ESP32-C5  |
| NM-CYD-C5              | **150.2 KH/s** | ESP32-C5  |

:::info
C5 지원은 지속적으로 해시레이트 개선을 받고 있습니다. 최신 수치는 [GitHub 릴리스 로그](https://github.com/NMminer1024/NMMiner#release-log)를 참조하십시오.
:::
