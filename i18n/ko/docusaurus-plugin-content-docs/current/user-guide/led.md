# LED Indicator

온보드 RGB 또는 상태 LED가 있는 보드는 이를 실시간 마이너 하트비트로 사용합니다.

| 상태             | LED 패턴                              |
| ---------------- | ------------------------------------- |
| WiFi 연결 중     | 호흡 효과                             |
| 마이닝 중        | 셰어 제출에 동기화된 느린 펄스        |
| 셰어 수락        | 빠른 밝은 깜박임                      |
| 셰어 거부        | 빠른 어두운 / 꺼짐 깜박임             |
| 오류             | 고정 색상 (모델에 따라 다름)          |

LED는 조명이 신경 쓰이는 경우 구성 포털에서 **비활성화**할 수 있습니다.
