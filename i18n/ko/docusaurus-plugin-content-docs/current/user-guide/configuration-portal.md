# Configuration Portal

구성 포털은 마이너가 AP 모드일 때 마이너 자체에서 제공되는 소형 캡티브 포털 스타일 웹 페이지입니다.

## 포털이 표시되는 경우

- 플래시 후 **최초 부팅**.
- **재연결 실패** — 마이너가 저장된 WiFi에 약 15초 내에 연결할 수 없으면 자동으로 AP 모드로 전환됩니다.
- **수동 강제** — 구성 버튼 길게 누르기 ([Buttons](./buttons.md) 참조).

## 접근 방법

1. 휴대폰이나 노트북에서 WiFi 설정을 엽니다.
2. 개방형 핫스팟 **`nmap-2.4g`**에 연결합니다.
3. 휴대폰에서는 캡티브 포털 브라우저가 자동으로 팝업됩니다. PC에서는 `http://192.168.4.1`을 엽니다.

[First Configuration](../getting-started/first-configuration.md)에 설명된 구성 폼이 표시됩니다.

## Save 동작

**Save**를 누르면 새 설정이 기록되고, 기기 내 저장소가 플러시될 때까지 기다린 후(10-15초) 재부팅됩니다. **이 시간 동안 전원을 끄지 마십시오** — 자체적으로 재시작되도록 두십시오.
