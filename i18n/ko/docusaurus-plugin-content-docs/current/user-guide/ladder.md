# Ladder — 글로벌 리더보드

**Ladder**는 NMMiner의 글로벌 리더보드입니다. 참여 마이너를 **역대 생성한 최고 셰어 난이도**로 순위를 매깁니다 — 해시레이트도, 셰어 수도 아닌, 당신의 역대 최고 기록입니다. 한 번의 행운의 순간이 $5짜리 ESP32를 ASIC 창고 위에 올릴 수 있습니다.

🌐 **공개 래더**: [https://ladder.nmminer.com/](https://ladder.nmminer.com/)
