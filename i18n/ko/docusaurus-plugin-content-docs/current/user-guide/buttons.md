# Buttons & Touch

다양한 보드에 단일 버튼(`BOOT`), 이중 버튼(`BOOT` + `USER`) 또는 터치 스크린이 있습니다. NMMiner는 세 가지 모두에 걸쳐 일관된 제스처를 제공합니다.

## 단일 버튼 보드

| 제스처        | 동작                                              |
| ------------- | ------------------------------------------------- |
| 싱글 클릭     | 화면 깨우기 (디스플레이가 있는 보드에서).         |
| 더블 클릭     | 다음 화면 페이지로 전환.                          |
| 길게 누르기   | 마이너 구성 진입 (AP 모드 강제).                  |

## 이중 버튼 보드

| 버튼    | 제스처        | 동작                                                    |
| ------- | ------------- | ------------------------------------------------------- |
| `BOOT`  | 싱글 클릭     | 화면 깨우기.                                            |
| `BOOT`  | 더블 클릭     | 화면 깨우기.                                            |
| `BOOT`  | 길게 누르기   | NVS에서 모든 저장된 상태 삭제 (빌드에서 활성화된 경우). |
| `USER`  | 싱글 클릭     | 화면 깨우기.                                            |
| `USER`  | 더블 클릭     | 다음 화면 페이지로 전환.                                |
| `USER`  | 길게 누르기   | 마이너 구성 진입 (AP 모드 강제).                        |

## 터치 스크린 보드

v2.0.01부터, 터치 보드(NM-TV-154, CYD 2.4 / 2.8 / 3.5, Waveshare 3.5 터치 등)는 **화면에서 직접** 제스처를 지원합니다:

| 제스처                        | 동작                                              |
| ----------------------------- | ------------------------------------------------- |
| 탭                            | 다음 화면 페이지로 전환 (화면이 절전 상태인 경우 깨우기도 함). |
| **10초간 길게 누르기**        | 마이너 구성 진입 (AP 모드 강제).                  |

페이지 순서는 [Screens](./screens.md), 길게 누른 후의 동작은 [Configuration Portal](./configuration-portal.md)을 참조하십시오.
