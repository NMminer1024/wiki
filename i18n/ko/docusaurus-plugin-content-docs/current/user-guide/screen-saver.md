# Screen Saver

구성 가능한 유휴 시간이 지나면 디스플레이가 화면 보호기 모드로 전환됩니다.

## 모드

| 모드     | 효과                                                                                       |
| -------- | ------------------------------------------------------------------------------------------ |
| `IMAGE`  | NM Monitor를 통해 업로드된 사용자 정의 이미지 슬라이드쇼. 부드러운 페이드 인 / 페이드 아웃. |
| `BLACK`  | 백라이트 끄기 (OLED는 최소 명암비로). 전력 절약 및 **절대 최고 해시레이트 제공**.           |

[NM Monitor](./nm-monitor.md)의 **Preferences** 섹션에서 모드를 실시간으로 전환할 수 있습니다 — 재부팅 필요 없음.

## 사용자 정의 이미지

NM Monitor의 **Screensaver** 페이지에서는 브라우저에서 직접 일반 PNG / JPG 파일을 업로드할 수 있습니다. 브라우저가 각 이미지를 NMMiner의 컴팩트한 `.ss` 형식(RGB565 + RLE)으로 변환하여 기기의 `/ss/` 폴더로 푸시합니다.

- 이미지는 **현재 화면 해상도**에 자동으로 맞춰집니다.
- 각 `.ss` 파일은 최대 **200 KB**입니다.
- 기기가 디스크 공간을 관리합니다 — FS가 가득 차면 가장 오래된 화면 보호기가 덮어쓰기됩니다.

스크립트로 업로드하고 싶다면? [API › Screensaver Upload](../api/screensaver-upload.md) 참조.

:::tip
절대 최고 해시레이트를 원한다면 **Black** 모드를 선택하십시오 — 디스플레이 새로고침은 CPU 사이클을 소모하며, 그 사이클은 해싱에 사용될 수 있습니다.
:::

## 깨우기

| 보드 유형      | 깨우기 제스처                      |
| -------------- | ---------------------------------- |
| 버튼 전용      | 아무 버튼이나 싱글 클릭.           |
| 터치 디스플레이 | 화면 아무 곳이나 탭.               |
