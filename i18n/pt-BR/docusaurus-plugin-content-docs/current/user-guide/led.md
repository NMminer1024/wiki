# Indicador LED

Placas com um LED RGB ou de status na placa o utilizam como um sinal de batimento cardíaco do minerador em tempo real.

| Estado                | Padrão do LED                                           |
| --------------------- | ------------------------------------------------------- |
| Conectando ao WiFi    | Respiração                                              |
| Minerando             | Pulsação lenta sincronizada com o envio de shares       |
| Share aceita          | Flash rápido e brilhante                                |
| Share rejeitada       | Flash rápido e fraco / apagado                          |
| Erro                  | Cor sólida (depende do modelo)                          |

O LED pode ser **desativado** no portal de configuração se você se incomodar com a luz.
