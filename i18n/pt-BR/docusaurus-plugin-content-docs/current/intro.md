# Wiki do NMMiner

Bem-vindo ao centro de documentação para usuários e integradores do **NMMiner**.

O **NMMiner** é um firmware de alto desempenho que transforma placas de desenvolvimento ESP32 comuns em mineradores Bitcoin (e DigiByte) via Stratum. Ele funciona nas famílias de chips ESP32-D0, ESP32-S3, ESP32-C3 e ESP32-C5, com taxas de hash de pico no dispositivo de:

- **ESP32-D0** — até **1.035 KH/s**
- **ESP32-S3** — até **400 KH/s**
- **ESP32-C3** — até **402 KH/s**
- **ESP32-C5** — até **150 KH/s**

## O que você pode fazer com o NMMiner

- Gravar uma placa ESP32 de baixo custo e começar a minerar na **pool SoloBTC do próprio NMMiner** em minutos — o firmware já vem pré-configurado.
- Usar o portal cativo no celular ou PC para alterar a pool, mudar o WiFi, definir a carteira e as opções de tela.
- Visualizar a taxa de hash ao vivo, URL da pool, estatísticas de shares, gráficos de preços de BTC / múltiplas moedas e clima no display do dispositivo.
- Abrir o **NM Monitor** em qualquer navegador (ele é servido pelo próprio minerador) para gerenciar configurações, enviar protetores de tela personalizados e acompanhar toda a frota local através do menu **Swarm**.
- Criar seus próprios dashboards ou automações utilizando a API HTTP documentada.
