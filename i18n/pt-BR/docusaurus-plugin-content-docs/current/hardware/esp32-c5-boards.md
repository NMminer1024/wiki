# Placas ESP32-C5

O ESP32-C5 é um chip RISC-V dual-band mais recente. O NMMiner fornece suporte, incluindo a placa de desenvolvimento dedicada NM-CYD-C5.

| Placa                  | Taxa de hash   | Chip      |
| ---------------------- | -------------- | --------- |
| ESP32-C5 DevKitC-1     | **150,2 KH/s** | ESP32-C5  |
| NM-CYD-C5              | **150,2 KH/s** | ESP32-C5  |

:::info
O suporte ao C5 continua recebendo melhorias de taxa de hash. Consulte o [registro de lançamentos do GitHub](https://github.com/NMminer1024/NMMiner#release-log) para os números mais recentes.
:::
