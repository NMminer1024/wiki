# Placas ESP32-C5

ESP32-C5 es un chip RISC-V de doble banda más reciente. NMMiner proporciona soporte incluyendo la placa de desarrollo dedicada NM-CYD-C5.

| Placa                  | Tasa de hash   | Chip      |
| ---------------------- | -------------- | --------- |
| ESP32-C5 DevKitC-1     | **150,2 KH/s** | ESP32-C5  |
| NM-CYD-C5              | **150,2 KH/s** | ESP32-C5  |

:::info
El soporte para C5 continúa recibiendo mejoras de tasa de hash. Consulte el [registro de lanzamientos de GitHub](https://github.com/NMminer1024/NMMiner#release-log) para las cifras más recientes.
:::
