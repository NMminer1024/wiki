# Indicador LED

Las placas con un LED RGB o de estado en placa lo usan como latido del minero en tiempo real.

| Estado                | Patrón de LED                                       |
| --------------------- | --------------------------------------------------- |
| Conectando WiFi       | Respiración                                         |
| Minando               | Pulso lento sincronizado con el envío de shares       |
| Share aceptada        | Destello brillante rápido                            |
| Share rechazada       | Destello tenue / apagado rápido                      |
| Error                 | Color fijo (depende del modelo)                      |

El LED puede **desactivarse** en el portal de configuración si le molesta la luz.
