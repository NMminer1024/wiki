# LED Indicator

Boards with an on-board RGB or status LED use it as a real-time miner heartbeat.

| State           | LED pattern                          |
| --------------- | ------------------------------------ |
| Connecting WiFi | Breathing                            |
| Mining          | Slow pulse synchronised to share submission |
| Share accepted  | Quick bright flash                   |
| Share rejected  | Quick dim / off flash                |
| Error           | Solid colour (model-dependent)       |

The LED can be **disabled** in the configuration portal if you mind the light.
