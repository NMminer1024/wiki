# Activate Licence

The licence pins a firmware image to a specific board. It is generated and returned by the flash tool.

## Steps

1. After flashing, the flash tool shows a long licence string. **Copy** it.
2. Power-cycle the board if it has not rebooted automatically.
3. Connect to the open hotspot `nmap-2.4g` and let the captive portal open (or visit `http://192.168.4.1`).
4. Paste the licence string into the **Licence** box and press the button.
5. On success, the page will display a confirmation and the miner will reboot into normal mode.
