# ESP32-C5 Boards

ESP32-C5 is a newer dual-band RISC-V chip. NMMiner provides support including the dedicated NM-CYD-C5 development board.

| Board                  | Hashrate       | Chip      |
| ---------------------- | -------------- | --------- |
| ESP32-C5 DevKitC-1     | **150.2 KH/s** | ESP32-C5  |
| NM-CYD-C5              | **150.2 KH/s** | ESP32-C5  |

:::info
C5 support continues to receive hashrate improvements. See the [GitHub release log](https://github.com/NMminer1024/NMMiner#release-log) for the latest figures.
:::
